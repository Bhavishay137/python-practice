## Overview

This repository contains my structured Python practice programs, organized in a logical sequence to reflect my learning progression. It focuses on building strong fundamentals in Python through step-by-step implementation of core concepts.

All programs are arranged in a clear and sequential order to maintain readability and structured growth.

---

## Learning Structure

This repository follows a progressive learning path where each file builds upon the previous concept.

### 01 – Hello World
Introduction to Python syntax and the basic print statement.

### 02 – Variables and Output
Understanding variable declaration, reassignment, and printing multiple values.

### 03 – Data Types
Exploration of Python’s built-in data types and usage of the type() function.

### 04 – Basic Input Handling
Introduction to taking user input from the console.

### 05 – Handling Multiple Inputs
Understanding how to process multiple values provided by the user.

### 06 – Type Casting
Converting input values into specific data types such as float and understanding explicit type conversion.
